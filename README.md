# 在线留言板

一个支持后台管理的在线留言板系统。

## 功能特性

### 前台功能
- 发布留言
- 查看所有留言
- 查看管理员回复

### 后台功能
- 管理员登录
- 修改密码
- 查看留言统计
- 回复留言
- 隐藏/显示留言
- 删除留言

## 部署到 Vercel

1. Fork 或 clone 本项目
2. 在 Vercel 中导入项目
3. 部署完成后，在 Vercel 后台添加自定义域名
4. 在域名服务商处配置 DNS 解析

## 默认管理员账号

- 用户名：admin
- 密码：admin123

⚠️ 部署后请立即修改默认密码

## 技术栈

- 纯 HTML + JavaScript
- Tailwind CSS
- AiPexBase BaaS 后端服务
